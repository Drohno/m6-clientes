Ejercicio Práctico

Realiza un programa sencillo en JavaScript que calcule la letra del DNI a partir del número de DNI introducido por pantalla (prompt). El resultado se deberá mostrar en un párrafo HTML.
La forma de calcular la letra consiste en dividir el número de DNI entre 23 y el resto de la división indica la posición de la letra en el siguiente array:
var letras = ['T', 'R', 'W', 'A', 'G', 'M', 'Y', 'F', 'P', 'D', 'X', 'B', 'N', 'J', 'Z', 'S', 'Q', 'V', 'H', 'L', 'C', 'K', 'E', 'T'];
Consideraciones:
• Se puede utilizar el método parseInt() para pasar un string a número entero.
• Se puede utilizar el método toUpperCase() para convertir un string a mayúsculas.
• Para conseguir el resto de una división se puede utilizar el operador %.
• Se deberá validar el número introducido para que se encuentre en el siguiente intervalo: [0,99999999].
• Se puede utilizar innerHTML para insertar texto en un párrafo HTML.
